### jlu drcom gtkmm

linux only

#### make

`$ git clone https://github.com/suransea/jlu-drcom-gtkmm.git`

`$ cd jlu-drcom-gtkmm`

`$ mkdir build`

`$ cd build`

`$ cmake ..`

`$ make`

`$ ./jlu-drcom`

#### dependencies

* boost

* gtkmm

* gtk

####

* 配置文件位置: ~/.config/drcom-client.conf

* 日志位置: ~/drcomclient.log
